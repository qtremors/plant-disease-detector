## [1.0.0] - 2026-01-12

### Added
- **ML Inference Engine**: Implemented MobileNetV2-based classification for apple diseases.
- **Django Web Interface**: Responsive dashboard for uploading and analyzing plant leaves.
- **Interactive Visuals**: Plotly-based confidence charts for detailed analysis.
- **Scan History**: SQLite tracking of all previous predictions and uploaded images.
- **uv Integration**: Modern dependency management and reproducible environments.

### Fixed
- Initial repository structure and configuration.
- Model loading singleton pattern in service layer.
